🎯 Baseline Lane 機制（每堂前 10 min）

16 位主管 AI 經驗 baseline 落差大（0 經驗 → ChatGPT 重度用戶）。Adult learning 第一原則 = respect prior experience。每堂前 10 min 分流：

🟢 Catch-up Lane（0 經驗 / 上週沒做作業者）— Young 帶 10 min 補上週最小成果。下堂直接接得上。
🔵 Standard Lane（多數主管）— 確認 Triage / SKILL / Take-1 等預習作業 OK，直接進入主題講解。
🟣 Stretch Lane（ChatGPT 重度用戶 / 已 own own SKILL.md）— 拿到「進階挑戰卡」，課堂上做 next-level task（例：W1 已寫 CLAUDE.md → 試 multi-agent CLAUDE.md / W5 SKILL.md 已寫 → 試 Anthropic skills repo PR）。

實作：W0 self-assessment 結果決定每人初始 lane。主管可換 lane — 過程中發現自己 lane 不對，Young 私下調整。

📝 W0 — 預作業週 (5/31 那週)

開課前 1 週，每位主管完成 5 題 self-assessment + 老帶新 pair 配對。

5 題 Self-Assessment（Google Form，10 min）

  1. 你用過幾種 AI 工具？（none / ChatGPT 試過幾次 / 每週用 / 每天用 / 付費過）
  2. 你寫過 prompt 嗎？（不知道是什麼 / 試過 / 會調整 / 有自己常用 template / 寫過長 system prompt）
  3. 你用 AI 解過真實工作問題嗎？（沒有 / 1 次 / 偶爾 / 每週 / 已是工作流一部分）
  4. 你部門有人在用 AI 嗎？（不知道 / 沒有 / 1-2 個試試 / 多數人 / 已內建工作流）
  5. 你對共學最期待什麼？（開放回答，1-3 句）

Pair 配對策略

Young 根據 self-assessment 做老帶新 pair（不 random）：經驗高的配 0 經驗，第 1 個月固定 pair，W1 + W3 + W5 同一搭檔。第 2 個月才換配對。
