## 兩兩 Swap 規則

每對 20 min × 2 輪：
- **5 min**: 我教
- **5 min**: 你問
- **5 min**: 你試
- **5 min**: 互回 feedback

跨部門配對優先（HR ↔ 編輯 / 行銷 ↔ 業務 / etc）。第 2 輪換 partner。
