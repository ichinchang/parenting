# W0 — 預作業週

> 開課前 1 週。10 min self-assessment + 預讀 Anthropic Claude Code 101 開頭。

## 對應 Anthropic 教材

[Claude Code 101](https://anthropic.skilljar.com/claude-code-101) — **What is Claude Code** + **How Claude Code works**（總長約 20 min）

## 5 題 Self-Assessment

填一張 Google Form（講師發），10 min：

1. 你用過幾種 AI 工具？（none / ChatGPT 試過幾次 / 每週用 / 每天用 / 付費過）
2. 你寫過 prompt 嗎？（不知道是什麼 / 試過 / 會調整 / 有自己 template / 寫過 system prompt）
3. 你用 AI 解過真實工作問題嗎？（沒 / 1 次 / 偶爾 / 每週 / 已內建工作流）
4. 你部門有人在用 AI 嗎？（不知道 / 沒 / 1-2 個 / 多數 / 已內建）
5. 你對共學最期待什麼？（開放回答，1-3 句）

→ 結果決定 W1 開始你進 **Catch-up / Standard / Stretch** 哪 lane。

## 預讀（給自己）

打開 [Claude Code 101](https://anthropic.skilljar.com/claude-code-101) → 看完前 2 個 module：
- **What is Claude Code**（5 min）
- **How Claude Code works**（15 min）

不一定要看完，0 經驗主管建議至少看 What is Claude Code。

## W0 裝機 session（93 min 線上）

W0 除了上面的預作業，還有一場 **93 分鐘線上裝機 session**，老師帶大家把環境設好、跑完第一次 Claude 對話。

**課前先做**（老師會發「課前安裝準備 checklist」）：
- 註冊 GitHub 帳號
- 裝好 Claude Code 桌面版
- 確認公司電腦能裝軟體（裝不起來先找 IT，不要等當天）

裝不起來不用慌，session 當天會留時間處理個案。

**session 內第一次對話**：用 [`對話啟動卡.md`](對話啟動卡.md) — 依你的角色（高層 / 內容創作 / 業務電商 / 人資財務設計）挑 prompt 範本，第一次體驗「派工給 Claude」。
