# W11 — Demo 發表（Parallel + Gallery Walk）

> 結業課。**3-3.5 hr 下午場**（不是午餐）。
> **對應 Anthropic 教材**: 整合演練 — Demo 必須呈現 Before / After / Skill or CLAUDE.md / **驗收方式** / **風險** / 下一步

## 共學目標

公開展示 **12 週 before/after** + 慶祝 + 對接 Q4 next step。

不再是 N 人連續 demo（認知過載）。改 **parallel demo + gallery walk**：4 組同時小場 demo，各組推 1 人上台 + 其他人作品 gallery walk。

## 內容流程

| 時段 | 內容 |
|------|------|

```
Slide 1: Before — 12 週前怎麼做
```
