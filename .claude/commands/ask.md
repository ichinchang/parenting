# /ask — 引導我問清楚

User 想問 Claude 但不知怎麼開口。`/ask` 後 Claude 問用戶 4 個問題引導：

1. **餵 context**: 你的工作背景 / 部門 / 角色 — 1-2 句
2. **給角色**: 你希望 Claude 扮演什麼角色幫你 — (顧問 / 屬下 / 對方 / 客戶代言人)
3. **給範例**: 你心中「理想回答」長怎樣 — 給 1 個範例
4. **給限制**: AI 不該怎樣 — (太長 / 太技術 / 用 emoji / 太正式)

收集 4 答案後，組合成完整 prompt 跑給 Claude。

## 對應教材

這是 **W1「主管派工 SOP」**的對話介面化版本。讓主管不用記 SOP 公式，跑 `/ask` 就能引導出完整 prompt。
